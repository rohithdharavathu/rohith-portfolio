## What I Built

### Agentic AI Learning System (Hackathon Winner)
Built an adaptive learning system that uses AI to personalize technical upskilling paths for engineers. This won the internal hackathon at Trianz.

The system:
- Assessed current skill levels through automated evaluation workflows
- Used a knowledge graph to model skill dependencies and learning paths
- Generated personalized learning plans using LLM-based planning
- Adapted recommendations based on completion and assessment signals
- Agentic architecture: specialized agents for assessment, planning, content curation, and progress tracking

This project sparked my deep interest in multi-agent architectures. The insight: a single LLM call gives you a good answer. A workflow of specialized agents making decisions at each step gives you a system.
